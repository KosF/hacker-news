# Hacker News API - Test App

An application to display ten stories in a random order from the list of best stories.

More information available at: https://github.com/HackerNews/API.

## Install

To install dependencies

```
npm install
```

## Run

To run app
```
npm start
```

## Test

To run test
```
npm test
```

## Build

To build app
```
npm run build
```
